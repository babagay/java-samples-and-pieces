[операторы >>>, >>]

https://stackoverflow.com/questions/2811319/difference-between-and

>>> - логическое смещение (-2 >>> 1 = 2147483647,  -4 >>> 28 = 15)
>>  - арифметическое смещение, сохраняет знак, если число отрицательное (-2 >> 1 = -1,  5 >> 2 = 1)

64 << 1 // 64 * 2^1 = 128
64 <<< 1 // error
64 >> 2 // 64 / 2^2 = 64 / 4 = 16
64 >>> 1 // 64 / 2^2 = 64 / 4 = 16
-1 >>> 1 // 2147483647
-2 >>> 1 // 2147483647
-1 >> 1 // -1
-2 >> 1 // -1
-2 << 1 // -2 * 2^1 = -4
-2 <<< 1 // Error(true)
10 << 1 // 10 * 2^1 = 20
10 << 2 // 10 * 2^2 = 40
10 << 3 // 10 * 2^3 = 80
